# NGSPacker

PSO2NGSのクライアントで読み込めるICE形式のファイルを、開いたり作成したりするためのプログラム。

このプログラムは、Shadowth117氏の[ZamboniLib](https://github.com/Shadowth117/ZamboniLib)を使用しています。
